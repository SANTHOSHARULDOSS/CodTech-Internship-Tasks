# CodTech IT Solutions Internship - Cybersecurity Tasks

This repository contains the four mandatory tasks completed during my Cybersecurity Internship at CodTech IT Solutions.

## ✅ Task 1: File Integrity Checker
- **Description:** A Python tool that monitors file changes using SHA-256 hashing.
- **Key Concepts:** Hashing (hashlib), File Monitoring, Integrity Verification.
- **Folder:** [TASK1](./TASK1)

## ✅ Task 2: Web Application Vulnerability Scanner
- **Description:** A Python-based scanner that detects SQL Injection vulnerabilities in web applications.
- **Key Concepts:** Web Requests, SQL Injection, BeautifulSoup, Automation.
- **Folder:** [TASK2](./TASK2)

## ✅ Task 3: Penetration Testing Toolkit
- **Description:** A multi-module toolkit containing a Port Scanner and a Password Strength Checker.
- **Key Concepts:** Socket Programming, Network Scanning, Password Complexity Analysis.
- **Folder:** [TASK3](./TASK3)

## ✅ Task 4: Advanced Encryption Tool
- **Description:** A secure file encryption and decryption tool using the Fernet symmetric key algorithm.
- **Key Concepts:** Cryptography, Fernet, Symmetric Encryption, File Handling.
- **Folder:** [TASK4](./TASK4)
